# 牛奶订购H5 - UI设计规范

## 设计理念

牛奶配送应用，传达新鲜、健康、便捷的感觉。

## 配色方案

| 颜色 | 色值 | 用途 |
|------|------|------|
| 主色 | #4A90D9 | 导航栏、按钮、主要交互 |
| 强调色 | #FF5500 | 价格、促销、购买按钮 |
| 辅助色 | #FF9900 | 购物车、待处理状态 |
| 背景色 | #F5F5F5 | 页面背景 |
| 文字色 | #333333 | 主要文字 |
| 次要文字 | #999999 | 说明文字 |

## 页面结构

### 1. 首页
- 顶部：搜索栏 + 分类入口
- 中部：商品列表（卡片式，2列网格）
- 底部：Tab导航

### 2. 商品详情页
- 顶部：商品图片轮播
- 中部：价格、名称、规格、描述
- 底部：数量选择 + 按钮栏

### 3. 购物车
- 商品列表（支持选择）
- 底部：合计 + 结算按钮

### 4. 订单确认页
- 收货地址
- 商品清单
- 费用明细

### 5. 订单列表
- Tab切换：全部/待支付/待配送/配送中/已完成
- 订单卡片展示

### 6. 个人中心
- 用户头像 + 信息
- 功能菜单列表

## 字体规范

| 用途 | 字号 |
|------|------|
| 页面标题 | 32rpx |
| 商品名称 | 28rpx |
| 价格 | 32rpx |
| 说明文字 | 24rpx |

## 交互规范

- 按钮点击有0.5s过渡效果
- 页面切换使用uni.navigateTo
- 操作反馈使用uni.showToast
